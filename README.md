# 8de backend server
* Docker installation required.

## 1. git clone
 ```
 git clone https://github.com/Silicon-Valley-Team-A/server.git
 ```

## 2. Docker image build
 ```
 docker-compose up --build
 ```

## 3. When the build finished, go to https://127.0.0.1
